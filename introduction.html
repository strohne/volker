Set custom item labels

Item labels are stored in the comment attribute of an item variable. When printing item tables, the labels are used. You can set custom or new labels by providing a tibble with item names in the first column and item labels in the second.


newlabels <- tribble(
  ~item_name, ~item_label,
  "cg_adoption_advantage_01", "Allgemeine Vorteile",
  "cg_adoption_advantage_02", "Finanzielle Vorteile",
  "cg_adoption_advantage_03", "Vorteile bei der Arbeit",
  "cg_adoption_advantage_04", "Macht mehr Spaß"
)

data %>% 
  set_item_labels(newlabels) %>% 
  tab_item_metrics(starts_with("cg_adoption_advantage_"))
