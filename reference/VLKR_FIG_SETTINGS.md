# Resolution settings for plots

Override with
`options(vlkr.fig.settings=list(html = list(dpi = 192, scale = 2, width = 910, pxperline = 15)))`.
Add a key for each output format when knitting a document. You can
override the width by setting vlkr.fig.width in the chunk options.

## Usage

``` r
VLKR_FIG_SETTINGS
```

## Format

An object of class `list` of length 2.
