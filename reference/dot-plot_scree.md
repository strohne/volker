# Helper function: scree plot

Helper function: scree plot

## Usage

``` r
.plot_scree(data, k = NULL, lab_x = NULL, lab_y = NULL)
```

## Arguments

- data:

  Dataframe with the factor or cluster number in the first column and
  the metric in the second.

- k:

  Provide one of the values in the first column to color points up to
  this value.

- lab_x:

  Label of the x axis

- lab_y:

  Label of the y axis

## Value

A vlkr_plot object
