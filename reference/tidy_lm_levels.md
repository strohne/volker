# Tidy lm results, replace categorical parameter names by their levels and add the reference level

Tidy lm results, replace categorical parameter names by their levels and
add the reference level

## Usage

``` r
tidy_lm_levels(fit)
```

## Arguments

- fit:

  Result of a [`lm`](https://rdrr.io/r/stats/lm.html) call.

## Value

A tibble with regression parameters.
