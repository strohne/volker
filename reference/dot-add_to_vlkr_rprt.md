# Add an object to the report list

Add an object to the report list

## Usage

``` r
.add_to_vlkr_rprt(obj, chunks, tab = NULL)
```

## Arguments

- obj:

  A new chunk (volker table, volker plot or character value).

- chunks:

  The current report list.

- tab:

  A tabsheet name or NULL.

## Value

A volker report object.
