# Insert a name-value-pair into an object attribute

Insert a name-value-pair into an object attribute

## Usage

``` r
.attr_insert(obj, key, name, value)
```

## Arguments

- obj:

  The object.

- key:

  The attribute key.

- name:

  The name of a list item within the attribute.

- value:

  The value of the list item.

## Value

The object with new attributes.
